Constructora Vial

Plataforma B2B web para la gestión integral de flotas y maquinaria. Solución enfocada en la optimización de recursos, rastreo geográfico, asignación de equipos y control de mantenimientos preventivos.

PHP Laravel 12 Blade Tailwind CSS MySQL

Sistema de Gestión - Constructora Vial

Proyecto integrador desarrollado para las cátedras de Programación y Práctica Profesionalizante, conceptualizado como una solución B2B para empresas prestadoras de servicios viales.

El sistema centraliza y optimiza la operativa de la empresa, permitiendo:

  • Gestión de Maquinaria y Obras: Desarrollo de CRUD completo para administrar flotas, registrar nuevas obras y gestionar las asignaciones de equipos a diferentes proyectos.
  • Geolocalización: Implementación de rastreo de ubicación por máquina mediante mapas interactivos (Leaflet) con geolocalización por provincias.
  • Mantenimiento Preventivo: Sistema de alertas automáticas que se disparan cuando una maquinaria supera los 500 km de uso, previniendo fallas y optimizando recursos.
  • Stack Tecnológico: Interfaz moderna y responsiva desarrollada con Laravel 12, Tailwind CSS y Flowbite en el frontend, respaldada por una base de datos MySQL.
